Polynômes du second degré

f(x) = ax² + bx + c  ·  a ≠ 0

Je veux retourner à l'acceuil cette page me fait peur
Index des cours de maths

Discriminant

Δ = b² − 4ac

Racines (si Δ ≥ 0)

x = (−b ± √Δ) / 2a

Sommet S(α ; β)

α = −b / 2a β = −Δ / 4a

Forme canonique

f(x) = a(x − α)² + β

Relations racines-coeff.

x₁ + x₂ = −b/a x₁ × x₂ = c/a

Forme factorisée (Δ > 0)

f(x) = a(x − x₁)(x − x₂)
